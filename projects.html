CukeBins

Description
CukeBins allows Cucumber to support step definitions written in C++. It relies on external testing frameworks such as GTest (Google C++ Testing Framework), CppSpec, Boost Test.
Web site
https://github.com/paoloambrosio/cukebins/wiki

Description
The Data-14 Project aimed to provide an open-source cross-platform CDP (Continuous Data Protection) solution for both server and desktop environments. It used file system filters to track file changes, and store them on a time-based hierarchical storage server. It started as a business idea, but most of the code was released as Open Source. The software is far from being complete.

http://www.data-14.org/
Developer site
