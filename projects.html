CukeBins

Description
CukeBins allows Cucumber to support step definitions written in C++. It relies on external testing frameworks such as GTest (Google C++ Testing Framework), CppSpec, Boost Test.
Web site

Description
The Data-14 Project aims to provide an open-source cross-platform CDP (Continuous Data Protection) solution for both server and desktop environments. It uses file system filters to track file changes, and store them on a time-based hierarchical storage server. It started as a business idea, but I released most of the code I developed as Open Source. The software is far from being complete.
Web site
http://www.data-14.org/
